h1. SDL iOS build script

A script creates a pseudo-framework which contains a universal library that can be used on a iOS and in the iOS simulator

h2. How to use

* From the Terminal, run:

<pre>
rake
</pre>

* Download and use "Minimal SDL iOS Application Template for Xcode 4":https://github.com/manifest1985/sdl-ios-templates
